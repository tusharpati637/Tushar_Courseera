# Code of Conduct

## Our Pledge
We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment include respect and collaboration.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing the code of conduct.

## Scope
This Code of Conduct applies within all project spaces.

## Enforcement
Instances of abusive behavior may be reported to the project team.

## Enforcement Guidelines
Warnings, temporary bans, or permanent bans may be used.

## Attribution
This Code of Conduct is adapted from the Contributor Covenant.
